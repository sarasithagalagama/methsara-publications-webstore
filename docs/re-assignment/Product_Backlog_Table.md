## Story Point Estimation Guide

- **1-2 points:** Very simple task (< 4 hours)
- **3 points:** Simple task with minimal complexity (4-8 hours)
- **5 points:** Moderate complexity requiring design decisions (12-16 hours)
- **8 points:** Complex task with multiple components (20-24 hours)
- **13+ points:** Very complex, should be broken down into smaller stories
